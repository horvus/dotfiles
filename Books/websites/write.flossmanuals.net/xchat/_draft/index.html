XChat: Introduction

XChat

XChat is a friendly interface for text chatting over the internet using Internet Relay Chat (IRC) networks.  It eliminates the need to learn the commands and syntax used by native IRC allowing you to use dialogs and menus to get around.


 

You can also use XChat for sending private messages and for sending files.

XChat is very stable and has a minimal and clean interface. You can install XChat on Windows and Ubuntu (or any form of Linux).
