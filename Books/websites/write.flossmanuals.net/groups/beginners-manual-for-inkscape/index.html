Just letting everyone know that I have started a new chapter on how to install Inkscape to a Mac in the book “Start Inkscape:”

 
http://write.flossmanuals.net/start-with-inkscape/_draft/_v/2.9/installing-inkscape/
 
This chapter has two sections: one on how to install Inkscape into a Mac, and one on how to install XQuartz into a Mac, and Inkscape needs XQuartz to run properly on a Mac.
