ACERCA DE ESTA (6ta) VERSIÓN

A year on from the 5th release, this release adds some exciting new sections as well as a number of chapter augmentations and necessary updates. Notable are Michael Gogins' Chapter on running Csound within a browser using HTML5 technology, Victor Lazarrini's and Ed Costello's explanations about Web based Csound, and a new chapter describing the pairing of Csound with the Haskell programming language.

Un año después de la quinta versión, en esta nueva entrega se agregan algunas nuevas interesantes secciones, actualizaciones que eran necesarias y el enriquecimiento de un número de capítulos. Resultan notables el capítulo de Michael Gogins sobre el funcionamiento de Csound dentro de un navegador usando la tecnología HTML5, las explicaciones de Victor Lazarrinis y Ed Costellos sobre Csound basado en la Web, y un nuevo capítulo que describe el maridaje de Csound con el lenguaje de programación Haskell.

Agradecemos a todos aquellos que colaboraron en esta versión.

Qué hay de nuevo en esta versión

  • Se agregó una sección acerca de la necesidad de inicializar explícitamente las variables-k para múltiples llamadas de un instrumento o UDO en el capitulo 03A Pasada de Inicialización y Realización (ejemplos 8-10).

  • Se agregó una sección sobre el bucle while/until en el capítulo 03C Estructuras de control.

  • Ampliación del capítulo 03D Tablas de Función, agregando descripciones de las GEN 08, 16, 19 y 30 

  • Pequeñas adiciones en el capítulo 03E Arrays.

  • Algunas adiciones y una nueva sección para ayudar a usar los diferentes opcodes (schedule, event, scoreline, etc.) en 03F Eventos en Vivo.

  • Se han añadido ejemplos y descripciones para los opcode powershape y distort en el capítulo 04 Síntesis de Sonido: Modelado de Onda.

  • Se amplió el capítulo 05A Envolventes, principalmente para incorporar descripciones de transeg y cosseg.

  • Added example to illustrate the recording of controller data to the chapter 07C Working with Controllers at the request of Menno Knevel. 

  • Se añadió un ejemplo para ilustrar la grabación de datos de un controlador en el capítulo 07C Trabajando con controladores, a petición de Menno Knevel. 

  • Se actualizó el capítulo 10B Cabbage, prestando atención a algunas de sus características más recientes.

  • El capítulo 10F Csound basado en la Web ahora tiene una descripción sobre cómo usar Csound vía UDP y también sobre pNaCl Csound (escrito por Victor Lazzarini). La sección sobre Csound como una biblioteca Javascript (usando Emscripten) en el mismo capítulo ha sido actualizada por Ed Costello.

  • Se reorganizó el capítulo 12A Acerca de la API de Csound para Csound6 y se agregó una sección sobre el uso de Foreign Function Interfaces (FFI) (escrito por François Pinot).

  • Se añadió el capítulo 12G sobre Csound y Haskell (escrito por Anton Kholomiov).

  • Se añadió el capítulo 12H sobre Csound y HMTL, también explicando el uso de Widgets HTML5 (escrito por Michael Gogins).

 

Los ejemplos de este libro se incluyen en el software CsoundQt (en Ejemplos -> Ejemplos del manual FLOSS). Aún más, los ejemplos que requieren de archivos externos ahora se encuentran listos para usar desde el inicio.

 

Si necesita consultar versiones anteriores de este manual, puede encontrarlas en http://files.csound-tutorial.net/floss_manual. En ese enlace también encontrará los csd y las muestras de audio utilizadas en esta versión.

 

   Berlin. Marzo del 2015.
   Iain McCurdy y Joachim Heintz.
