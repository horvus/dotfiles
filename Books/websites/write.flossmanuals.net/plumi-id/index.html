Plumi

Plumi adalah Sistem Manajemen Konten Piranti Lunak Bebas yang dirancang untuk keperluan video-sharing, yang berbasis pada Plone dan diproduksi kolektif oleh EngageMedia. Plumi memungkinkan Anda untuk menciptakan situs video-sharing milik Anda sendiri; dengan menginstal Plumi pada server web yang Anda miliki, Anda dapat menggunakan sejumlah besar fungsi yang ada untuk pengiriman video dan penciptaan komunitas

Fitur-fitur yang terdapat di dalamnya adalah video podcasting, server-side flash/ogg transcoding, lisensi bebas, suatu pemunculan alur kerja yang sangat canggih dan pengunggahan file berukuran besar melalui FTP. Anda juga dapat menggunakan berbagai fungsi bawaan Plone lainnya termasuk news, events, comments dan profil user.

 

 

Manual ini mencakup panduan dasar berbagai fitur bagi para pengguna awam, bagaimana menggunakan situs tool administrasi, dan instalasi serta konfigurasi dari situs jaringan Plumi yang baru pada web server Linux. Bagian-bagian pada instalasi dan konfigurasi memerlukan pengetahuan dasar seperti yang dicantumkan di bawah ini.

Terminologi Utama

Berbagai dokumen eksternal dirujuk pada manual ini – seringkali digunakan terminologi "Plone" sebagai kebalikan dari "Plumi". Hal ini terjadi karena Plumi berdasar pada sistem manajemen konten Plone, dan sebagian besar dokumen yang terdapat pada web secara umum ditulis untuk Plone, dan tidak secara spesifik untuk Plumi.

Bagi Para User

  • Vodcast atau video podcast – RSS-berbasis konten sindikasi, termasuk file video.
  • Dashboard – panel kontrol dasar yang dimiliki oleh user situs Plumi untuk mengakses berbagai fungsi yang ada termasuk profil, pilihan pribadi dan kontribusi situs.
  • FTP – (File Transfer Protocol) adalah suatu cara untuk mentransfer file, bukan dengan cara mengunggah ataupun mengunduh melalui browser web browser, tetapi dengan menggunakan FTP-client, yang memiliki fitur-fitur seperti mengunggah atau mengunduh. 
  • Plumi – aplikasi web video-sharing yang berbasis pada Plone/Zope, yang menjadi alasan ditulisnya manual ini.
  • Plone – Sistem Manajemen Konten Piranti Lunak Bebas yang berbasis pada Zope.
  • Zope – aplikasi server dan basis data
  • Python – bahasa pemograman dinamis yang digunakan dalam sebagian besar penulisan Plone/Plumi, yang digunakan dalam sebagian besar aplikasi domain. Python seringkali dibandingkan dengan Perl, Ruby, Scheme atau Java.
  • ZMI (Zope Management Interface) – suatu interface untuk mengkonfigurasi Zope, termasuk kemampuan untuk menambah dan mengkonfigurasi situs Plone (misalnya).
  • Transcode Daemon – kerangka kerja video transcoding dan layanan jaringan kerja, yang mengonversi berbagai file video dengan menggunakan ffmpeg, yang diintegrasikan ke dalam Plone.
  • Skin atau Theme – rancangan visual website Plone, termasuk template laman, CSS dan gambar.
  • Product – paket Python untuk Zope, sejenis modul piranti lunak yang dapat digunakan dalam Plone.
  • Package – suatu istilah yang lebih umum, akan tetapi dalam dunia Zope sering dimaknai sebagai sesuatu yang tidak terdapat dalam Products name space.
  • Egg – suatu cara untuk mendistribusikan paket. Mendukung manajemen dependensi otomatis: dependensi dapat diunduh dari PyPI/The Cheese Shop.
  • GenericSetup – suatu cara untuk mengkonfigurasi situs Plone dengan menggunakan file XML yang deklaratif.
  • Policy Product – suatu pola dimana sebuah proyek memiliki produk tunggal yang, pada saat penginstalasian, menginstal seluruh dependensi dan kustomisasi dalam satu langkah.
