Handbrake: BasicUSe

Penggunaan Dasar

Ketika kamu memulai Handbrake kamu harusnya melihat layar dibawah.



Sekarang saya akan mengarahkanmu melalui penggunaan Handbrake paling dasar – menggunakan Handbrake untuk mengubah sebuah DVD menjadi file mp4 yang dapat didistribusikan di internet.

Pertama-tama kamu harus memasukkan sebuah disk DVD ke dalam komputermu.

Kemudian klik tombol ‘Browse’ di sebelah bagian layar yang tertulis “Source”.




Setelah men-klik tombol ini, sebuah window seharusnya keluar seperti dibawah.




Gulir kebawah untuk melokasikan DVD di daftar ini dan klik di tombol plus sebelahnya:




Kamu sekarang dapat melihat file-file di dalam DVD. Ini mungkin tidak masuk akal bagi yang belum pernah membuka DVD dengan cara ini sebelumnya. Kita tidak perlu untuk tahu banyak mengenai struktur file DVD untuk menggunakan Handbrake. Yang perlu kamu ketahui hingga langkah ini adalah direktori VIDEO_TS mengandung informasi file videonya. Jadi pilihlah direktori “VIDEO_TS” (jangan klik dobel, soroti saja folder ini) dan klik “OK”. Window berikut ini seharusnya terlihat:



Window diatas membutuhkan waktu sebentar untuk hilang karena Handbrake menganalisa struktur DVD.

Kamu mungkin melihat bagian dibawah source window yang tertulis ‘Title:’ dan “Bab:”. Jika kamu ingin, jangan sentuh bagian ini, agar settingannya menjadi default. Ini akan encode semua informasi di dalam disk DVD.

Memilih Judul Bab

Kamu mungkin ingin memilih hanya satu ‘Title’/judul (biasanya satuan dilm di dalam DVD) atau hanya pilih ‘Bab’ tertentu (biasanya bagian dari film yang diciptakan untuk membuat navigasi DVD semakin mudah).

Jika ini kasusnya kamu dapat memilih Titles/Judul dan Chapters/Bab di bagian “Source” dari window default Handbrake.



Kamu dapat melihat panjang dari judul-judul berbeda (film) hingga bisa mengidentifikasikan yang mana yang ingin dipilih.

Setel Output File dan Format

Pertama-tama kita butuh untuk menyetel direktori output untuk file yang kita akan ciptakan. Klik di tombol “Browse” di bagian Destination di window:



Sekarang kita akan melihat sebuah browser file.



Pilih sebuah direktori dikomputermu dimana mp4 akan diciptakan dan disimpan. Beri nama filenya di dalam kotak ‘File Name’ dan biarkan setelan default mp4 di dalam kotak ‘Save as type’. Kamu dapat mengubahnya menjadi format file lain jika kamu inginkan, tapi untuk pengenalan dasar Handbrake kita akan membiarkannya menjadi mp4.

Lalu kamu dapat memulai proses encoding DVDnya menjadi file mp4 dengan klik di tombol besar “Start” di bagian layar paling atas, seperti terlihat di bawah ini:

Ketika proses encoding sedang terjadi seharusnya terlihat sebuah layar hitam yang memberitahumu prosesnya sedang berlangsung dan berapa lama prosesnya.



Jika kamu belum melihat window seperti ini sebelumnya ini dikenal sebagai Window DOS atau command line/barisan perintah.

Ketika prosesnya telah selesai gunakan sebuah program seperti VLC player untuk mengetes file yang telah kamu buat. Kamu mungkin ingin menguploadnya ke internet ke servis video sharing seperti Archive.org atau Blip.tv.
