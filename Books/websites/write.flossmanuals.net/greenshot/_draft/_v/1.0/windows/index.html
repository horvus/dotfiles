Software name : Greenshot

Homepage : http://getgreenshot.org/  

Software version used for this installation : 1.2.10.6  

Operating system used for this installation : Microsoft Windows 10

Downloading Greenshot

The latest stable version of Greenshot for Microsoft Windows can be downloaded from : http://getgreenshot.org/downloads/ 

The latest stable version at the time of writing this document is Greenshot 1.2.10.6

     

Click on the “Latest Stable Windows” link. This should automatically launch the download of Greenshot-INSTALLER-1.2.10.6-RELEASE.exe. Once the download is complete the file should be near the bottom of your browser (in this case the browser is Chrome):

or you can browse directly to your “Downloads” directory and look for the most recent downloaded file:

 

 

Double click on the icon or click on “Open” on the downloads window to open this file and begin installing Greenshot

You should now see a Setup Wizard:

Choose your langauge and click on "OK"

The next step asks you to read and accept the License Agreement:

After you have read the agreement click on "I accept the agreement" then click on "Next >"

You should now see an Information window:

Click on "Next >"

You will now be prompted to select the type of installation you wish to complete:

It is recommended to do a Default Installation. To do so, click on "Next >"

You should now see a window like this: 

Click the box beside "Start Greenshot with Windows start" if you want Greenshot to start when you turn your computer on. Otherwise click on "Next >"

Greenshot is now ready to be installed: 

Click on "Install"

You should now see this window: 

After Greenshot finishes installing a setup window will appear:

Click on "Finish" to complete the installation and start Greenshot.
